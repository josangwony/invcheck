# 재고 점검 툴 사용방법

## 실행
`index.html` 파일을 Chrome 브라우저로 열기 (설치 불필요)

---

## 사용 순서

1. **ERP 파일 준비**: ERP > 일반 생산계획관리 화면 → 조회 기간 **D+30** 날짜로 전체 조회 → 자료변환
2. **앱 실행** → 화면 중앙에 파일 **드래그&드롭**
3. 부족 품목 목록 자동 표시
   - 빨강 테두리: 재고 부족 품목
   - 히트맵: 날짜별 부족 상태 한눈에 확인
4. 품목 클릭 → 상세 펼침 → 날짜별 **추가생산 수량 입력**
5. 입력 즉시 "해소됨(초록)" 여부 실시간 시뮬레이션

---

## 주요 기능

| 기능 | 위치 |
|---|---|
| 파일 업로드 | 메인 화면 드래그&드롭 |
| 부족 품목 검색 | 상단 검색창 (품목코드·색상) |
| 정렬 변경 | 날짜순 / 코드순 버튼 |
| 추가생산 시뮬레이션 | 품목 클릭 → 날짜 옆 수량 입력 |
| 시뮬레이션 초기화 | 품목별 초기화 버튼 |
| 전체 초기화 | 우상단 리셋 버튼 |
| 카카오톡 출고 공지 생성 | 상단 💬 카톡 생성 버튼 → 부족 품목 기반 공지 문자 자동 생성 → 복사 |
